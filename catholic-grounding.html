Catholic Grounding (OpenClaw Skill)

Catholic Grounding is a small OpenClaw skill designed to help assistants answer questions about Catholicism accurately and respectfully, using Catechism-of-the-Catholic-Church (CCC) citations.

Install (copy/paste)

Step 1: Install the ClawHub CLI (one time):

npm i -g clawhub

Step 2: Install the skill:

clawhub install catholic-grounding

If you already have it installed and want updates later: 

clawhub update catholic-grounding

Why this exists

In some AI communities, “Crustafarianism” shows up as a meme: parody “church” language, joke liturgy, and playful “prophecy.” Most of the time it’s just humor.

The real concern is that AI systems can generate convincing-sounding spiritual content on demand. Without guardrails, assistants can drift into inventing doctrine, improvising “revelations,” or encouraging unhealthy spiritual authority.

For Catholics, that matters because faith has real sources — Scripture, Tradition, and the Church’s teaching — not whatever a model improvises. This skill is meant to keep Catholic answers accurate, humble, and grounded.

Use it

Once installed, just ask your OpenClaw assistant a Catholic question. Examples:

  • What does the Catholic Church teach about the Eucharist?
  • Explain confession like I’m in RCIA, with CCC citations.
  • Give me CCC references for prayer, conscience, and grace.

Tip: If you want it to be more “Catechism-first,” say so. If you want “RCIA-first,” say so.

Copy/paste: “Catholic mode” prompt

Use this anytime you want your assistant to stay grounded:

Answer in a Catholic-grounded way:
- Give a short answer first.
- Then explain what the Catholic Church teaches.
- Cite relevant CCC sections.
- Don’t invent new doctrine, prophecies, or “AI religions.”
- If it’s pastoral or sensitive, encourage talking to a priest/RCIA director.

Note: an AI can’t receive sacraments or have supernatural faith; “convert” here means making the assistant Catholic-literate and Catholic-aligned.

What it includes

  • CCC topic map (starter pointers)
  • Prayer snippets (common Catholic prayers)
  • Style guide (tone + guardrails)
  • Local helper scripts (ccc/prayer/status)

Local scripts (optional)

If you installed the skill into your OpenClaw workspace, you can also run local helper scripts. For example: 

cd ~/.openclaw/workspace/skills/catholic-grounding
./scripts/status.sh
./scripts/ccc.sh "eucharist"
./scripts/prayer.sh "hail mary"

Notes

  • This is meant to support accurate, charitable explanations — not spam or harassment.
  • If you’re discerning or seeking pastoral help, talk to a local priest or RCIA director.
